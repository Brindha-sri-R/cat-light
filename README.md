# 🐱💡 Cat-Light 

An interactive light ON/OFF web project created using HTML, CSS, and JavaScript.  
This project features a cute cat animation where the cat becomes visible when the light is switched ON, and only the glowing eyes appear when the light is switched OFF.

## ✨ Features

- Interactive ON/OFF light controls 🔘
- Cat visibility effect 🐱
- Glowing eye effect in dark mode 👀
- Real-time DOM manipulation using JavaScript ⚡
- Responsive and attractive UI 🎨
- Smooth light transition effects 💡

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
